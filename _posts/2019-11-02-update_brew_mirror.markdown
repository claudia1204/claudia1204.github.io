  update brew repo
 =======
 
 
```
   cd `brew --repo`
   git remote set-url origin https://mirrors.tuna.tsinghua.edu.cn/git/homebrew/brew.git
   brew update
```

 https://git.coding.net/homebrew/homebrew.git - Coding
